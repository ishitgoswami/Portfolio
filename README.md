## 🛠️ Technology Stack
This portfolio is built with modern web technologies:
* **Framework:** React.js (Vite)
* **Styling:** Tailwind CSS V4 
* **Animations:** Motion (Framer Motion)
* **Icons:** Lucide React
* **Contact Form:** Web3Forms API
* **Language:** TypeScript

## ✨ Features
* **Modern UI:** Glassmorphism design with a dynamic warm orange, cyan, and emerald neon color palette.
* **Smooth Animations:** Scroll-triggered reveal animations and interactive hover effects.
* **Responsive Layout:** fully optimized for mobile, tablet, and desktop viewing.
* **Working Contact Form:** Integrated with Web3Forms to send messages directly to email without requiring a backend server.
* **Dynamic Sections:** Hero, About, Skills, Projects, Training, Certificates, Achievements, and Academics.

## 💻 Running Locally

To run this project on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ishitgoswami/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View in browser:**
   Open `http://localhost:5173` in your favorite web browser.
