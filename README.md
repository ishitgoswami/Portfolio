# Tania Bhattacharyya - Developer Portfolio 🚀

A modern, interactive, and responsive personal portfolio website showcasing my skills, projects, and professional journey as a developer with a focus on AI/ML and Full-Stack Development.

## 🌟 Highlights & Features

- **Modern UI/UX**: Built with **React** and styled using **Tailwind CSS** with custom glassmorphism effects and neon accents.
- **Smooth Animations**: Powered by **Framer Motion** for scroll-reveal animations, hover effects, and dynamic modals.
- **Interactive Project Showcase**: Expandable project cards with detailed descriptions, tech stacks, and direct GitHub links.
- **Coding Profiles Integration**: Live statistics and direct links to competitive programming profiles (LeetCode, GeeksforGeeks, Coding Ninjas).
- **Working Contact Form**: Integrated with Web3Forms to send emails directly from the frontend without needing a backend server.
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile viewing.

## 🛠️ Tech Stack

- **Frontend Framework**: React 18 + Vite
- **Styling**: Tailwind CSS (Utility-first CSS framework)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## 📂 Key Sections

1. **About / Hero**: Introduction and quick resume access.
2. **Skills & Expertise**: Categorized display of technical proficiencies (Python, Java, React, PyTorch, etc.).
3. **Projects**: Detailed case studies of projects such as *Karshika Sakhi* (AI Farming Companion), *WasteNot* (Smart Kitchen Assistant), and *Student Management System*.
4. **Professional Journey**: Details of my freelance Python backend and web development experience.
5. **Coding Profiles**: Competitive programming achievements and stats.
6. **Certifications**: Relevant industry credentials (IBM, Oracle, Infosys).
7. **Academics**: Educational background and performance.
8. **Contact**: Functional email form and social media links.

## 🚀 Run Locally

To run this project on your local machine, follow these steps:

**Prerequisites:** Node.js installed on your system.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/taniabhat/Portfolio.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd Portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open in browser:**
   Open [http://localhost:5173](http://localhost:5173) to view the application.
